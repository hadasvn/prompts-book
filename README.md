# ספר פרומפטים חכם

אפליקציית React לוקאלית לגמרי (בלי שרת) — הופכת את 20 הפרומפטים הקבועים לכלי עבודה חי:
פרופיל עסק גלובלי, מילוי אוטומטי של placeholders, ניתוח תיאור עסק, ייבוא וסיווג פרומפטים
חיצוניים, ושמירה לקובץ מקומי / לתיקיית Drive.

כל הנתונים (פרופיל, פרומפטים שמורים, הגדרות) נשמרים ב-`localStorage` של הדפדפן — אין שרת ואין
מסד נתונים חיצוני.

## חשוב — למה הקוד רץ מ-C:\ ולא ישירות מתוך Drive
Google Drive (הכונן G:\) הוא כונן וירטואלי שלא מסתדר טוב עם `node_modules` (אלפי קבצים קטנים) —
ניסיון להתקין תלויות ישירות בתוך תיקיית ה-Drive גרם לשיבוש קבצים. לכן:

- **הפיתוח בפועל** רץ מהעותק המקומי: `C:\Users\hadas\dev\prompts-book`
- **קוד המקור מסונכרן** (בלי `node_modules`) גם לתיקיית ה-Drive, כדי שיהיה נגיש דרך Git/לצפייה.
- כדי להריץ את האפליקציה תמיד עובדים מול העותק ב-`C:\Users\hadas\dev\prompts-book`, לא מתוך Drive.
